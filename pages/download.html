Welcome to Password Manager


How to download

Step 1:

Download the given Setup File file for the software to run.

Step 2:

Install the given setup file. Having the shortcuts is recommended.

Step 3:

Once installed the shorcut for the executable will be present onDesktop andStart Menu namedPassword Manager.

Step 4:

Intially you will be prompted to have an Admin Password to access all the Username and Passwords stored. With Admin Password, Admin Mail ID is also required in case Admin Password is forgotten.

Step 5:

Once you have completed all these steps, you are ready to store your username and password with the service provider.

Step 6:

You can store as many passwords as you want.
