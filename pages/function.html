Welcome to Password Manager


Functionality

Once you open the program, you need to enter your admin password to view further options. These further options include retrieving a password from your saved list, storing a new one, updating an existing one, deleting a password, show the list of websites or apps for which you have stored a password, or update your admin password or the recovery email.

# Command What it does
1 Exit Quits the program
2 Get Password Retrieve's the password
3 Store Password Stores the password
3 A Custom Password Creates a password of 16 characters length and Stores the password
3 B User Password Requires a password to be given by the user.
4 Update Password updates the exixting password to a new one
5 Delete Password Delete's the password the is not in use anymore
6 Show services Show's the services and the usernames for which password is stored
7 Update Admin Password Update's the Admin password
8 Update Admin Mail ID Update's the Recovery Mail ID

To access these options, all you need to do is type the respective abbreviation mentioned along with the choice to enter the respective menu.
